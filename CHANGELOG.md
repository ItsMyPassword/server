# Changelog

All notable changes to the Keyfount server are documented here. The version of
each release is driven by the `release:*` label on its `develop → main` pull
request; the git tag `vX.Y.Z` is the source of truth and matches the published
container image tag.

## v0.0.1 — 2026-05-28

Initial versioned container release of the Keyfount sync server, published to
GHCR for linux/amd64 and linux/arm64.
